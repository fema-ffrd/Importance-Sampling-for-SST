SSTImportanceSampling.stormdepthprocessor.StormDepthProcessor

class SSTImportanceSampling.stormdepthprocessor.StormDepthProcessor(data, *, zero_pad_edges=True)[source]

Bases: object

Evaluate watershed-averaged storm depths for translated storm centers, then compute weighted exceedance probabilities per realization.

Inputs

The constructor expects an object data (e.g., your Preprocessor output or a loader) providing:

  • cumulative_precip:

    • xarray.DataArray with dims ("storm_path","y","x"), or

    • xarray.Dataset containing variable "cumulative_precip", or

    • dict[str, xarray.DataArray(y,x)] mapping storm id → 2-D field (mm).

  • watershed_gdf : GeoDataFrame of the target watershed (same grid CRS).

  • storm_centers : DataFrame with original centers (columns ["storm_path","x","y"]).
